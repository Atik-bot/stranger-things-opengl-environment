# Stranger Things OpenGL Environment

## Overview

A 2D interactive graphics environment inspired by Stranger Things, developed using OpenGL and GLUT. The project demonstrates fundamental computer graphics concepts including custom drawing algorithms, 2D transformations, animation, and scene rendering.

---

## Features

- Interactive 2D environment
- OpenGL and GLUT rendering
- Midpoint Circle Algorithm implementation
- Object transformations
- Animation effects
- Multi-object scene composition
- Real-time rendering

---

## Technologies Used

- C++
- OpenGL
- GLUT
- Computer Graphics

---

## Graphics Concepts Implemented

- Midpoint Circle Algorithm
- Translation Transformations
- Animation Techniques
- Procedural Scene Generation
- Object Rendering
- Interactive Graphics
